# 通用舆情监控平台 v2.0

**多平台数据抓取 + Kimi AI 智能分析**

支持 Steam、小黑盒、B站、TapTap、知乎等多个平台，一键生成 AI 舆情报告。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🎮 **多平台监控** | Steam、小黑盒、B站、TapTap、知乎 |
| 🤖 **Kimi AI 分析** | 智能总结、情感分析、风险识别 |
| 📊 **统一 Dashboard** | 跨平台数据对比、趋势分析 |
| ⚠️ **风险预警** | 负面舆情自动识别、分级提醒 |
| 💡 **运营建议** | AI 生成的改进建议 |

## 🚀 快速部署

### 1. 准备环境变量

在 Vercel 添加以下环境变量：

```bash
# MongoDB Atlas（必须）
MONGODB_URI=mongodb+srv://username:password@cluster0.xxx.mongodb.net/universal_monitor?retryWrites=true&w=majority

# Kimi AI（必须）
KIMI_API_KEY=sk-your-kimi-api-key
KIMI_MODEL=moonshot-v1-8k
```

**获取 Kimi API Key：**
1. 访问 https://platform.moonshot.cn/
2. 注册/登录账号
3. 创建 API Key
4. 复制到环境变量

### 2. 部署到 Vercel

```bash
# 1. 推送代码到 GitHub
git add .
git commit -m "Universal monitor v2.0"
git push

# 2. Vercel 自动部署
# 或访问 https://vercel.com/new 手动部署
```
